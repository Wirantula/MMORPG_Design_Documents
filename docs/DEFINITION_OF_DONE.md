# Definition of Done
A story is done only when all of these are true:

1. Scope and acceptance criteria are satisfied.
2. Dependencies are respected and no contract violations remain.
3. Lint, typecheck, and tests pass.
4. Documentation and ADRs are updated where applicable.
5. Observability impact is included (at least one relevant log/metric/event).
6. Security and abuse implications were reviewed.
7. Product Owner or designated reviewer accepted the story.
